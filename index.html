GöPDF WebApplikation
Im Laufe meines Projektseminars "Entwicklung von WebApplikation" an der Universität Göttingen, habe ich eine WebApp mit dem Thema "Entwicklung einer REST-API zur Anfrage von nutzer-generierten Formularen im Kontext der auto­matisierten Dokumentenerstellung an Universitäten" geschrieben.

In simpleren Worten sind die Hauptanforderungen der Seite:
  • Erstellen von Dokumenten mit dynamischen Platzhaltern in einem Editor
  • Das Dokument soll, nach einem Freigabeprozess, über eine REST-API als PDF abrufbar sein
  • Die Platzhalter sollen im API-Request durch die Angabe von Parametern ersetzt werden können

Zusätzlich gibt es auch Nebenanforderungen wie:
  • Rechteverwaltung (Redakteur | Admin )
  • Nutzer-, API-, Dokumentenverwaltung, API-Logs
  • Frontend: Dashboards, Universitäts Design, Responsive

Die Seite wird auf einem Projektserver der Universität Göttingen gehostet.
Weitere Informationen können auf der Hilfeseite eingesehen werden.

ChatBot
Anleitung:
Der ChatBot benutzt OpenAI's "GPT-3.5 Language Model" um eine Antwort zurückzugeben. Schreibe eine Frage in die linke Box und klicke auf das Chatsymbol. Außerdem ist es mit OpenAI's "DALL·E" möglich sich über ein bestimmtes Thema realistische Bilder erstellen zu lassen. Wähle dafür in der Selection "Draw an Image" aus.
Gedächtnisspiel
Anleitung:

Anleitung:
Das Programm ruft Länderinformationen von der "GeoDB Cities" API ab und stellt sie strukturiert dar. Suche im linken Bereich nach einem Land und klicke es an, um Informationen zu erhalten. Anschließend können mit einem Klick auf das Pluszeichen jeweils 10 Regionen des Landes geladen werden. Die API ist beschränkt auf eine Abfrage pro Sekunde (Ein Klick ist eine Abfrage).
